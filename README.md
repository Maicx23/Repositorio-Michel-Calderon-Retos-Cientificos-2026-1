## Reto o Experimentación

El tema elegido es ***dinámica***, específicamente la dinámica de cuerpo rígido.

*¿En que medida afecta la naturaleza de una superficie el coeficiente de resistencia a la rodadura de un cuerpo rigido y cómo se puede usar esto a nuestro favor?*

El propósito del reto es calcular el coeficiente de ***rodadura*** entre un cilindro y una superficie, y con ello, analizar el comportamiento y crear relaciones prácticas.
